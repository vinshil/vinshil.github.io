头像设置

1、头像图片放入主题文件中**themes/主题/source/images**中

2、修改主题配置，找到**Sidebar Avatar**字段，图片路径放在url后面就行了

代码高亮

1、站点配置中找到 highlight下面的auto_detect设为ture

2、主题配置highlight_theme改为你喜欢的

  • night
  • night eighties
  • night blue
  • night bright

选项 默认值 功能描述
id 必须值 歌曲 id / 播放列表 id / 相册 id / 搜索关键字
